## Parameters
```t=[[num1,"object_type1"],[num2,"object_type2"]]```
* **num1** - integer, from 0 to 12.
* **object_type1** - string, in folowing list.
* **num2** - integer, from 0 to 12.
* **object_type2** - string, in folowing list.
